The salary calcluator is a project aimed at calculating your salary income based on monthly outgoings such as mortgage, rent, gym, food, bills and all other outgoings you might have on a monthly basis. This project uses totally custom JS and Bootstrap 3.3.5 with the Fezvrasta Bootstrap Material UI applied with some minor changes.
